# Don't Sink

这是一个使用Svelte框架和Babylon.js引擎开发的前端小游戏。

## 简介

在这个游戏中，玩家的目标是避免让船只沉没。游戏由纯前端技术实现，目前没有后端支持。

## 安装和使用

1. 克隆项目仓库：

   ```bash
   git clone https://github.com/Oloongbee/dont_sink.git
   ```
2. 切换项目文件夹：

   ```bash
   cd fe
   ```
3. 安装依赖：

   ```bash
   npm install
   ```
4. 启动开发服务器：

   ```bash
   npm run dev //默认8080端口
   ```

## 技术栈

* Svelte
* Babylon.js
* JavaScript
